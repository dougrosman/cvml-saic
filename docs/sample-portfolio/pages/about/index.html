This is a sample portfolio web page for the Spring 2020 Computer Vision and Machine Learning Class at SAIC.

If you would like to download the code used to create this portfolio page, it is all contained in a folder you can download here.

To edit this code, open this folder in VS Code (File, Open, then select the 'sample-portfolio' folder) and then turn your Live Server on (the "Go Live" button at the bottom right of your screen). You can use the html and css pages I've created here as starter templates for your own portfolio site. Feel free to copy and paste any of this code into the files in your own repository. Just remember to link your files correctly!
